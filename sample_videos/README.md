# Sample Videos

Place traffic video files in this directory for real video mode.

## Supported formats
- `.mp4` (recommended)
- `.avi`
- `.mkv`

## Free sources for traffic videos
- [Pexels Traffic Videos](https://www.pexels.com/search/videos/traffic/)
- [Pixabay Traffic Videos](https://pixabay.com/videos/search/traffic/)

## Usage
1. Download a traffic video
2. Place it in this folder as `traffic_sample.mp4`
3. Set `VIDEO_SOURCE=sample_videos/traffic_sample.mp4` in `.env`
4. Set `SIMULATION_MODE=false` in `.env`
5. Restart the backend

> **Note**: The system works fully in simulation mode without any video files.
